# 学生成绩管理系统（练手项目）
    本系统主要包含 课程设置、学生管理、成绩管理、系统管理 这几大部分，具体功能如下：
+ 课程设置：为学生和教师对各上课班级公布成绩的属性设置。

+ 学生管理：为每个学生提供考试信息、成绩查看与学分总数，绩点查看等服务。

+ 成绩管理：为系统管理员和教师提供管理各个班级学生成绩的管理设置。

+ 系统管理：为系统管理员提供管理学生和教师与课程的各种权限与服务。


>用到的东西：  
    JDK1.8.  
    MySQL（在Bitnami的容器中）  
    IntelliJ IDEA（开发所需）  
    JFormDesigner（开发时使用的是idea插件版的）
